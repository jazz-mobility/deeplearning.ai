## Key Concepts

### Agentic AI
AI systems that can:
- Set and pursue goals
- Plan multi-step workflows
- Iterate and improve outputs
- Adapt to changing conditions
- Learn from feedback

### Design Patterns
- **ReAct**: Reasoning + Acting
- **Plan-and-Execute**: Upfront planning with execution
- **Reflection**: Self-evaluation and iteration
- **Tool Use**: Integrating external capabilities

### Evaluation Approaches
- Task success rates
- Step efficiency
- Quality of intermediate outputs
- Human evaluation
- A/B testing
